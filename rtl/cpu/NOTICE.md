# Vendored third-party cores — provenance & license

## mc6809i.v
- **Author:** Greg Miller — "Cycle-Accurate 6809 Core", Copyright (c) 2016 Greg Miller.
- **Upstream:** https://github.com/cavnex/mc6809
- **Obtained from:** the JTFRAME copy at `modules/jtframe/hdl/cpu/mc6809i.v` in
  Jotego's jtcores.
- **Modifications:** none — file copied verbatim, original header preserved.
- Driven directly (no JTFRAME `jtframe_sys6809` wrapper) so this project does not
  inherit the wrapper's dependencies. E/Q clock enables are generated by
  `rtl/breakthru_clocks.sv`; the external data-in mux, RAM, and interrupt logic
  live in `rtl/brkthru_main.sv`.
